> **DRV8353**驱动**BSC035N10NS5**

## 配置总览

| 地址 | 寄存器         | 说明                |
| ---- | -------------- | ------------------- |
| 0x02 | Driver Control | PWM等控制配置寄存器 |
| 0x03 | Gate Drive HS  | 高侧MOS驱动寄存器   |
| 0x04 | Gate Drive LS  | 低侧MOS驱动寄存器   |
| 0x05 | OCP Control    | 过流保护寄存器      |
| 0x06 | CSA Control    | 电流采样寄存器      |

可以看到，整个DRV8353可读写的寄存器大概分为四类：

配置类（0x02 控制寄存器）；

驱动类（0x03 0x04 驱动电流寄存器）；

保护类（0x05 过流保护寄存器）；

采样类  (0x06 电流采样寄存器) 。

我们计算的重点就落在后面三类上。

---

## 0x02 — Driver Control — 控制寄存器

| 字段     | 说明                 | 编码                                     |
| -------- | -------------------- | ---------------------------------------- |
| OCP_ACT  | 过流后是否三相全关断 | 0=单桥 OCP 关断，1=三相全关断            |
| DIS_GDUV | VCP/VGLS 欠压保护    | 0=使能，1=禁用                           |
| DIS_GDF  | 栅极驱动故障检测     | 0=使能，1=禁用                           |
| OTW_REP  | 过温警告上报         | 0=不上报，1=上报 nFAULT                  |
| PWM_MODE | PWM 输入模式         | 00=6x PWM，01=3x PWM，10=1x PWM，11=独立 |
| 1PWM_COM | 1x PWM 整流方式      | 0=同步整流，1=异步整流                   |
| 1PWM_DIR | 1x PWM 方向          | 与 INHC 输入 OR                          |
| COAST    | 全桥高阻态           | 写 1 全三相桥进入高阻态                  |
| BRAKE    | 三相低侧刹车         | 写 1 三相低侧导通                        |
| CLR_FLT  | 清除锁存故障         | 写 1 清除（自动复位为 0）                |

## 0x03 — Gate Drive HS — 高侧MOS驱动寄存器

| 字段       | 说明                   | 编码                       |
| ---------- | ---------------------- | -------------------------- |
| LOCK       | 寄存器读写锁定         | 110=锁定，011=解锁读写     |
| IDRIVEP_HS | 高侧栅极拉电流（开管） | 0000-1111 对应 50-1000 mA  |
| IDRIVEN_HS | 高侧栅极灌电流（关管） | 0000-1111 对应 100-2000 mA |

## 0x04 — Gate Drive LS — 低侧MOS驱动寄存器

| 字段       | 说明                   | 编码                                                       |
| ---------- | ---------------------- | ---------------------------------------------------------- |
| CBC        | 周期过流清除           | OCP_MODE=01 时：0=tRETRY 后清除，1=新 PWM 或 tRETRY 后清除 |
| TDRIVE     | 峰值驱动电流持续时间   | 00=500 ns，01=1 µs，10=2 µs，11=4 µs                    |
| IDRIVEP_LS | 低侧栅极拉电流（开管） | 0000-1111 对应 50-1000 mA                                  |
| IDRIVEN_LS | 低侧栅极灌电流（关管） | 0000-1111 对应 100-2000 mA                                 |

## 0x05 — OCP Control — 过流保护寄存器

| 字段      | 说明             | 编码                                         |
| --------- | ---------------- | -------------------------------------------- |
| TRETRY    | 过流后重试等待   | 0=8 ms，1=50 µs                             |
| DEAD_TIME | 半桥死区         | 00=50 ns，01=100 ns，10=200 ns，11=400 ns    |
| OCP_MODE  | VDS 过流响应     | 00=锁存关断，01=自动重试，10=仅报告，11=禁用 |
| OCP_DEG   | VDS 过流去毛刺   | 00=1 µs，01=2 µs，10=4 µs，11=8 µs       |
| VDS_LVL   | VDS 过流检测阈值 | 0000-1111 对应 0.06-2.0 V                    |

## 0x06 — CSA Control — 采样电流寄存器

| 字段        | 说明                 | 编码                                     |
| ----------- | -------------------- | ---------------------------------------- |
| CSA_FET     | 电流采样正输入选择   | 0=SPx（分流电阻），1=SHx（MOS 源极）     |
| VREF_DIV    | 采样偏置参考电压     | 0=VREF（单向），1=VREF/2（双向）         |

| CSA_GAIN    | 电流采样放大增益     | 00=5，01=10，10=20，11=40 V/V            |
| DIS_SEN     | Sense 过流检测使能   | 0=使能，1=禁用                           |
| CSA_CAL_ABC | 三相电流通道偏置校准 | bit4/3/2 写 1 分别短接 A/B/C 相输入      |
| SEN_LVL     | Sense 过流检测阈值   | 00=0.25 V，01=0.5 V，10=0.75 V，11=1.0 V |

## 计算过程

### 驱动

#### 前置条件

在配置高低侧驱动电流之前，首先要判断DRV8353到底适不适合你的应用场景，会不会是大炮打蚊子/狗骑吕布。

思路：假设DRV8353以最低档电流（维持MOS管为打开状态时的电流）驱动MOS，在给定的时间内能否将MOS驱动至完全打开/关断。

公式：

$$
I_{VCP/VGLS} > Q_g \times f_{PWM}   （六步换相）
$$

或

$$
I_{VCP/VGLS} > 3 \times Q_g \times f_{PWM} （FOC控制）
$$

$I_{VCP/VGLS}$：$I_{VCP}$为电荷泵最小驱动电流，$I_{VGLS}$为稳压器最小驱动电流。由于半桥驱动电路中的高侧驱动会使用电荷泵进行自举，所以衡量高侧的驱动能力往往用电荷泵的驱动电流描述。而下管通常不需要自举，所以直接使用的稳压器进行驱动。二者都是用于描述驱动能力大小，作用是一样的，只是严谨一些区分开。

$Q_{g}$：MOS 管栅极总电荷，描述从关断到完全导通需向栅极注入的电荷（$Q = I \cdot t$）。可分解为：

$$
Q_g \approx Q_{gs} + Q_{gd} + Q_{other}
$$

$Q_{gs}$：MOS管**栅极-源极电容电荷（G-S）**，栅极驱动电路最先驱动的电容的电荷。

$Q_{gd}$：MOS管**栅极-漏极电容电荷（G-D）**，又称米勒电荷。在 GS 极电容充电的过程中，$V_{GS}$ 逐渐升高直到到达 $V_{GS(th)}$，此时 MOS 管会逐渐开始导通，$R_{DS}$ 会逐渐下降，$V_D$ 随着 $R_{DS}$ 的下降而下降，由于 $V_{GD} = V_G - V_D$，GD 极之间会存在电位差，于是驱动器会转为向 GD 极之间的 Miller 电容 $C_{gd}$ 充电，本来流给 $C_{gs}$ 的电流被偷走给 $C_{gd}$ 充电，直到 $C_{gd}$ 充满。这个过程中 G 极电压不会有特别大的变化，在 MOS 管导通的过程中就像一个平台一样，所以又称这个过程为 Miller 平台。在这个过程中，MOS 管大概率卡在导通跟关断之间的线性区，发热量巨大。

$Q_{other}$：MOS管剩余电荷。MOS管从米勒平台到完全导通需要的电荷量。

$f_{PWM}$：控制MOS管开关的最大PWM频率。

从目标MOS（笔者这里是**BSC035N10NS5**）的数据手册可知 ${Q_g}$ = 70nC：

{% asset_img 1-1.png "Qg 数据手册摘录" %}

而笔者所用的SVPWM的频率 ${f}$ = 40kHZ；

于是就有：

$$
3 \cdot f \cdot Q_g = 3 \times 40\ \text{kHz} \times 70\ \text{nC} = 3 \times 0.0028\ \text{A} = 8.4\ \text{mA} < I_{VCP} / I_{VGLS}\ (25\ \text{mA})
$$

（DRV8353 具备驱动该 MOS 的能力。）

#### 初步计算

灌 / 拉电流：

$$
\frac{Q_{gd}}{T_r} < I_{drive\_p}
$$

从目标MOS（笔者这里是**BSC035N10NS5**）的数据手册可知 $Q_{gd} = 14\ \text{nC}$：

{% asset_img 1-2.png "Qgd 数据手册摘录" %}

$T_r$ 取 笔者所用SVPWM的频率 （${f}$ = 40khz）的分辨率 1/2125 （也就是MCU定时器的计数周期，又称自动重装载值）；

所以有：

$$
T_r = \frac{1}{40\text{k} \times 2125} \approx 1.1764 \times 10^{-8}\ \text{s} \approx 11.8\ \text{ns}
$$

拉电流：

$$
I_{drive\_p} = \frac{14\ \text{nC}}{1.1764 \times 10^{-8}\ \text{s}} \approx 1.19\ \text{A}
$$

考虑到灌电流（关断）需要更快速度（$T_n = 2T_f$），所以：

$$
I_{drive\_n} = I_{drive\_p} \times 2 = 2.38\ \text{A}
$$

#### 修正计算（不那么激进）

考虑到上述驱动参数过于激进（这里 $T_r$ 仅 11 ns），再提供一组不那么激进的参数，$T_r$ 取 40 kHz 下 PWM 分辨率的 **5 倍**：

$$
T_r = \frac{1}{40\text{k} \times 2125} \times 5 = 5.88 \times 10^{-8}\ \text{s} \approx 60\ \text{ns}
$$

重新计算如下：

| 参数                            | 计算                                     | 结果   |
| ------------------------------- | ---------------------------------------- | ------ |
| $I_{drive\_P1}$（拉电流下限） | $\dfrac{14\ \text{nC}}{60\ \text{ns}}$ | 233 mA |
| $I_{drive\_P2}$（拉电流上限） | $\dfrac{14\ \text{nC}}{30\ \text{ns}}$ | 466 mA |
| $I_{drive\_N1}$（灌电流下限） | $\dfrac{14\ \text{nC}}{30\ \text{ns}}$ | 466 mA |
| $I_{drive\_N2}$（灌电流上限） | $\dfrac{14\ \text{nC}}{15\ \text{ns}}$ | 933 mA |

#### 最终选取

| 参数              | 取值             | 对应时间                                                         |
| ----------------- | ---------------- | ---------------------------------------------------------------- |
| 拉电流（IDRIVEP） | **400 mA** | $T_r = \dfrac{14\ \text{nC}}{0.4\ \text{A}} = 35\ \text{ns}$   |
| 灌电流（IDRIVEN） | **900 mA** | $T_f = \dfrac{14\ \text{nC}}{0.9\ \text{A}} = 15.5\ \text{ns}$ |

对应寄存器 0x03 / 0x04：`IDRIVEP = 0110`（400 mA），`IDRIVEN = 0111`（900 mA）。

### 保护

#### VDS 保护

取目标 MOS 极限工况（140℃，50 A，Vgs = 10 V）下的 $R_{DS}$，从目标 MOS 数据手册查表得：

$$
R_{DS}(T_j = 140°C,\ I_{ds} = 50A,\ V_{gs} = 10V) \approx 5\ \text{mΩ（典型值）}
$$

考虑到实际过程中假设的 $I_{ds} = 100\ \text{A}$，$R_{DS}$ 会更糟，所以这里 $R_{DS}$ 取最大值 **6.25 mΩ**。

于是：

$$
V_{ds\_ocp} = I_{max} \cdot R_{DS} = 100\ \text{A} \times 6.25\ \text{mΩ} = 0.625\ \text{V}
$$

这里取 **0.6 V**。为了避免尖峰毛刺影响，**OCP_DEG 取最大值 8 µs**。

对应寄存器 0x05：`VDS_LVL = 1001`（0.6 V），`OCP_DEG = 11`（8 µs）。

#### 采样电阻保护

$$
V_{sen\text{-}ocp} = I_{max} \cdot R = 100\ \text{A} \times 2\ \text{mΩ} = 0.2\ \text{V}
$$

这里取 **0.25 V**（没有更小的了）。

对应寄存器 0x06：`SEN_LVL = 00`（0.25 V）。

#### 死区

40 kHz 下 PWM 分辨率是 **11 ns**（以 2125 为 TIM 一次计数周期），一个 PWM 周期为 **25 µs**（25000 ns）。

取 **100 ns** 死区，空间不大，不会压缩 PWM 空间。

对应寄存器 0x05：`DEAD_TIME = 01`（100 ns）。

---

### 采样

已知采样电阻为 **2 mΩ**。

由约束关系 $R = \dfrac{V_0}{A_v \cdot I}$，$V_0 = (V_{vref} - 0.25) - \dfrac{V_{vref}}{2} = 1.4\ \text{V}$ 得：

| 增益$A_v$ | 采样电流上下限$I = \dfrac{V_0}{R \cdot A_v}$ |
| ----------- | ---------------------------------------------- |
| 5 V/V       | $\pm 140\ \text{A}$                          |
| 10 V/V      | $\pm 70\ \text{A}$                           |
| 20 V/V      | $\pm 35\ \text{A}$                           |
| 40 V/V      | $\pm 17.5\ \text{A}$                         |

考虑到测试过程中转矩经常突破 50 N·m，换算下来 $I_q$ 也有 **30 A** 了，故 $A_v = 20\ \text{V/V}$ 时裕量处于紧张状态，故取 **$A_v = 10\ \text{V/V}$**。

对应寄存器 0x06：`CSA_GAIN = 01`（10 V/V），`VREF_DIV = 1`（以 Vref 为中心，双向电流）。
